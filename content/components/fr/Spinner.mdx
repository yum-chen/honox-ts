# Introduction

Un composant utilisé pour indiquer un état de chargement.

# Utilisation

<DemoPreview>
  <Spinner size="md" label="Loading..." />
</DemoPreview>

```tsx
import { Spinner } from "../components/ui";

export default function MyPage() {
  return <Spinner size="md" label="Loading..." />;
}
```

## Constructeur de pages CMS

Ce composant est disponible sous forme de bloc `spinner` dans [Page Builder](/docs/PageBuilder) (`content/pages/*.json`) :

```json
{
  "type": "spinner",
  "size": "lg",
  "label": "Loading"
}
```

# Propriétés

| Propriété | Type | Description |
| --- | --- | --- |
| `class` | `string` | Classes CSS personnalisées. |

| `size` | \`"xs" \ | "sm" \ | "md" \ | "lg" \ | "xl" \ | "2xl" \ | "inherit"\` | The size of the spinner. |

| `label` | `string` | Une étiquette accessible aux lecteurs d’écran. |

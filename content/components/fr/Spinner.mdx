# Introduction

A component used to indicate a loading state.

# Utilisation

<DemoPreview>
  <Spinner size="md" label="Loading..." />
</DemoPreview>

```tsx
import { Spinner } from "../components/ui";

export default function MyPage() {
  return <Spinner size="md" label="Loading..." />;
}
```

## Générateur de page CMS

Ce composant est disponible en tant que bloc `spinner` dans le [Générateur de page](/docs/PageBuilder) (`content/pages/*.json`) :

```json
{
  "type": "spinner",
  "size": "lg",
  "label": "Loading"
}
```

# Propriétés

| Propriété | Type | Description |
| --- | --- | --- |
| `class` | `string` | Classes CSS personnalisées. |

| `size` | \`"xs" \ | "sm" \ | "md" \ | "lg" \ | "xl" \ | "2xl" \ | "inherit"\` | The size of the spinner. |

| `label` | `string` | An accessible label for screen readers. |

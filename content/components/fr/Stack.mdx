Une primitive de mise en page qui peut être utilisée pour créer une pile verticale ou horizontale d'éléments.

## Utilisation

<DemoPreview>
  <Stack direction="row" gap="4">
    <div style={{ padding: "0.75rem 1rem", background: "var(--colors-blue-3)", borderRadius: "8px" }}>Item 1</div>
    <div style={{ padding: "0.75rem 1rem", background: "var(--colors-blue-3)", borderRadius: "8px" }}>Item 2</div>
    <div style={{ padding: "0.75rem 1rem", background: "var(--colors-blue-3)", borderRadius: "8px" }}>Item 3</div>
  </Stack>
</DemoPreview>

```tsx
import { Stack } from '../components/ui/stack'

export const Example = () => (
  <Stack direction="row" gap="4">
    <div>Item 1</div>
    <div>Item 2</div>
    <div>Item 3</div>
  </Stack>
)
```

### Constructeur de pages CMS

Ce composant est disponible sous forme de bloc `stack` dans [Page Builder](/docs/PageBuilder) (`content/pages/*.json`) :

```json
{
  "type": "stack",
  "direction": "horizontal",
  "gap": "4",
  "children": [
    { "type": "text", "content": "Item 1" },
    { "type": "text", "content": "Item 2" }
  ]
}
```

## Propriétés

| Propriété | Type | Défaut | Description |
| --- | --- | --- | --- |
| `direction` | `flex-direction` | `column` | La direction de flexion de la pile. |
| `gap` | `spacing` | `8px` | L'écart entre les éléments de la pile. |
| `align` | `align-items` | - | L'alignement des éléments dans la pile. |
| `justify` | `justify-content` | - | La justification des éléments dans la pile. |

## Réactivité

Le composant `Stack` prend en charge les valeurs réactives pour tous ses accessoires.

```tsx
<Stack direction={{ base: 'column', md: 'row' }} gap={{ base: '4', md: '8' }}>
  {/* Elements will be stacked vertically on mobile and horizontally on desktop */}
</Stack>
```

Un composant qui permet aux utilisateurs de saisir plusieurs balises sous forme de liste.

## Utilisation

<DemoPreview>
  <div style={{ width: "100%", maxWidth: "320px" }}>
    <TagsInput label="Skills" defaultValue={["React", "Hono"]} />
  </div>
</DemoPreview>

### Constructeur de pages CMS

Ce composant est disponible sous forme de bloc `tagsInput` dans [Page Builder](/docs/PageBuilder) (`content/pages/*.json`) :

```json
{
  "type": "tagsInput",
  "label": "Amenities",
  "defaultValue": ["WiFi", "Fireplace", "Hot Tub"]
}
```

## Propriétés

| Propriété | Type | Défaut | Description |
| --- | --- | --- | --- |
| `value` | `string[]` | - | Balises actuelles. |
| `defaultValue` | `string[]` | - | Balises initiales. |

## Configuration du CMS

```yaml
- label: TagsInput
  name: tagsInput
  widget: object
  fields:
    - { label: Label, name: label, widget: string, required: false }
    - { label: Default Tags, name: defaultValue, widget: list, required: false }
```

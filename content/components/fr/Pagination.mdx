Un composant pour naviguer dans plusieurs pages de contenu.

## Utilisation

<DemoPreview>
  <Pagination count={100} pageSize={10} />
</DemoPreview>

### Constructeur de pages CMS

Ce composant est disponible sous forme de bloc `pagination` dans [Page Builder](/docs/PageBuilder) (`content/pages/*.json`) :

```json
{
  "type": "pagination",
  "count": 100,
  "pageSize": 10
}
```

## Propriétés

| Propriété | Type | Défaut | Description |
| --- | --- | --- | --- |
| `count` | `number` | - | Nombre total d'articles. |
| `pageSize` | `number` | `10` | Nombre d'éléments par page. |
| `defaultPage` | `number` | `1` | La page initiale à afficher. |

| `type` | \`'button' \ | 'link'\` | `'button'` | Whether to use buttons or links for navigation. |

| `interactive` | `boolean` | `true` | S'il faut activer l'interactivité côté client. |

## Configuration du CMS

```yaml
- label: Pagination
  name: pagination
  widget: object
  fields:
    - { label: Count, name: count, widget: number }
    - { label: Page Size, name: pageSize, widget: number, default: 10 }
    - { label: Default Page, name: defaultPage, widget: number, default: 1 }
    - label: Type
      name: type
      widget: select
      options: [button, link]
      default: button
```

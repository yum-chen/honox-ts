This is the Notes app. Every note is a plain Markdown file under `content/notes/`, edited through the CMS at `/admin`.

- Pin a note to keep it at the top
- Give it a color to spot it at a glance
- Add tags to group related notes
- Archive it instead of deleting when you're done with it
